# Código de Conducta — Tótem USM

## Nuestro compromiso

Este proyecto busca ser un espacio de colaboración respetuoso e inclusivo. Todos los colaboradores —estudiantes, profesores, desarrolladores y personal USM— deben tratarse con respeto.

## Comportamiento esperado

- Usar lenguaje inclusivo y respetuoso
- Aceptar crítica constructiva con apertura
- Enfocarse en lo que es mejor para el proyecto y la comunidad USM
- Mostrar empatía hacia otros contribuidores

## Comportamiento inaceptable

- Comentarios ofensivos, discriminatorios o acoso de cualquier tipo
- Publicar información privada de otros sin permiso
- Conducta que razonablemente se consideraría inapropiada en un entorno profesional

## Reporte

Si presencias o experimentas comportamiento inaceptable, repórtalo al equipo responsable del proyecto.

Este código de conducta está adaptado del [Contributor Covenant](https://www.contributor-covenant.org/), versión 2.1.
